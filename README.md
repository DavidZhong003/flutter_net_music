# flutter_net_music

仿版网易云,个人学习Flutter项目,仅供学习用,请勿用于商业用途.

- 构建前需要搭建[服务后台](https://github.com/Binaryify/NeteaseCloudMusicApi)。

- 在lib下创建host.dart文件。

- 声明后台地址Host,`const String HOST = "http://你的后台地址:你的后台端口";`
